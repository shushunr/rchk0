# Wrap summary and raw issue outputs

Wrap summary and raw issue outputs

## Usage

``` r
issue_envelope(df_summary, df_raw, idx = NULL, label = NULL)
```

## Arguments

- df_summary:

  A summary data.frame

- df_raw:

  A raw data.frame

- idx:

  Optional index

- label:

  Optional label

## Value

A list with `summary` and `raw`
