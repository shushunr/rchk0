# Create a READ_ME worksheet in the output issue tracker

Create a READ_ME worksheet in the output issue tracker

## Usage

``` r
create_readme_sheet(wb, study_name = "", all_check, simple_check)
```

## Arguments

- wb:

  openxlsx workbook

- study_name:

  character, e.g., "I7P-MC-DSAF"

- all_check:

  data.frame from spec sheet "ALL_DATASETS"

- simple_check:

  data.frame from spec sheet "SINGLE_DATASETS"

## Value

wb
