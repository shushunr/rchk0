# Check for duplicate visits with the same date

Check for duplicate visits with the same date

## Usage

``` r
compute_duplicate_visit_date(df, ds, visit_date_col, visit_label_col, id_cols)
```

## Arguments

- datasets_pool:

  Named list of datasets

- wb:

  Workbook

- target_vars:

  Optional

- dataset_no:

  Datasets to exclude

- visit_info_df:

  Visit info

- output_tab:

  Output tab

- custom_code_list:

  Named list of custom check functions
